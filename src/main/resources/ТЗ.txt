Необходимо реализовать систему хранения информации о студентах.
Студенты обучаются на 3 ступенях: бакалавриат, магистратура и аспирантура.
Для всех студентов необходимо хранить данные:
- ФИО
- Дата рождения
- Дата поступления на направление
- Направление подготовки (буквенный префикс)

Для бакалавров необходимо хранить номер аттестата о среднем образовании.
Для магистров необходимо хранить номер диплома о получении степени бакалавра.
Для аспирантов необходимо хранить тему диссертации.

Название группы, в которой учится студент, меняется каждый год и составляется по шаблону: "<буквенный код ступени> + <буквенный префикс> -
<порядковый номер года обучения на данном направлении>". Например, для бакалавра 3 курса по специальности "Прикладная математика и
информатика" номер группы будет Б-ПМИ-3. Для магистра 1 курса по той же специальности: М-ПМИ-5 (он обучается на ней 5-й год).
